# 📊 AutoEDA Dashboard

AutoEDA Dashboard is a Streamlit-based web application that performs automatic Exploratory Data Analysis (EDA) on uploaded CSV datasets.

Users can upload datasets and instantly generate:
- dataset summaries
- missing value analysis
- statistical insights
- correlation heatmaps
- histograms
- automated EDA reports

---

# 🚀 Features

- Upload CSV datasets
- Automatic EDA report generation
- Dataset summary table
- Missing value analysis
- Correlation heatmap
- Interactive histogram visualization
- Simple and clean dashboard UI

---

# 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- ydata-profiling

# ▶️ Installation

Clone the repository:

```bash
git clone <your-github-repo-link>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
